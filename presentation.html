Xây dựng Agentic RAG
Chuyển từ linear RAG sang Self-RAG với LangGraph. Thiết kế toàn bộ 5-node graph: Preprocess, Retrieve, Rerank, Generate, Check Answer. Tích hợp vòng lặp Transform Query tự động tái truy vấn khi không đủ evidence.
LangGraph Self-RAG AgentState LangSmith
Generation + Citation
Grounded prompt chỉ bám evidence. Citation validation kiểm tra chunk_id / source / section / url. Detect ngôn ngữ từ câu hỏi gốc (Viet/Anh). Parse JSON structured output. Streaming response.
Grounded Prompt Citation Validation Streaming
Guardrails
Phát hiện câu hỏi ngoài tài liệu (not_found). Ngăn model hallucinate khi không có evidence. Validation: nếu không có valid citation → trả về not_found thay vì chọn citation sai.
not_found detection Hallucination guard
UI + Evaluation
Frontend Next.js: citation click xem source, source debug viewer, PDF/chunk mapping. Evaluation runner Excel: Recall@5, MRR@5, RAGAS (faithfulness, relevancy, precision, recall). LangSmith trace toàn bộ graph.
